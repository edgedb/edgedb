=====================
edgedb server restart
=====================

Restart an EdgeDB instance.

.. cli:synopsis::

     edgedb server restart <name>


Description
===========

``edgedb server restart`` is a terminal command for restarting an
EdgeDB server instance.


Options
=======

:cli:synopsis:`<name>`
    The EdgeDB instance name.
