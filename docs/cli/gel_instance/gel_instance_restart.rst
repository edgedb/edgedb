====================
gel instance restart
====================

Restart a |Gel| instance.

.. cli:synopsis::

     gel instance restart <name>


Description
===========

:gelcmd:`instance restart` is a terminal command for restarting an
|Gel| instance.

.. note::

    The :gelcmd:`instance restart` command is not intended for use with
    self-hosted instances.


Options
=======

:cli:synopsis:`<name>`
    The |Gel| instance name.
