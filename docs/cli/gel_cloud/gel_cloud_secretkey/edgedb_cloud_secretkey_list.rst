========================
gel cloud secretkey list
========================


List existing secret keys

.. cli:synopsis::

    gel cloud secretkey list [<options>]

.. note::

    This command works only if you have already authenticated using
    :ref:`ref_cli_gel_cloud_login`.

Options
=======

:cli:synopsis:`--json`
    Output results as JSON
