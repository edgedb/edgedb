=======================
edgedb instance restart
=======================

Restart an EdgeDB instance.

.. cli:synopsis::

     edgedb instance restart <name>


Description
===========

``edgedb instance restart`` is a terminal command for restarting an
EdgeDB instance.


Options
=======

:cli:synopsis:`<name>`
    The EdgeDB instance name.
