===============
Getting Started
===============

Welcome to EdgeDB!

Here we cover the steps needed to set up EdgeDB and start using
it. This is a good starting point if you're new to EdgeDB.

1. Start with the :ref:`installation <ref_tutorial_install>`.
2. Set up your :ref:`database and schema <ref_tutorial_createdb>`.
3. Learn the basics about :ref:`EdgeQL <ref_tutorial_queries>`.
4. Make your data accessible via :ref:`GraphQL <ref_tutorial_graphql>`.

For more examples check out our `interactive tutorial
<https://tutorial.edgedb.com/>`_.
